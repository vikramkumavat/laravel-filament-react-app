# Screenshots Directory

This directory contains screenshots for the project documentation.

## Adding Screenshots

To add a screenshot to the README:

1. Save your screenshot image file in this directory (`docs/images/`)
2. Use a descriptive filename (e.g., `admin-dashboard.png`)
3. Update the README.md file to reference the image path

### Example

If you save an image as `admin-dashboard.png` in this directory, reference it in README.md as:

```markdown
![Admin Dashboard](docs/images/admin-dashboard.png)
```

## Supported Formats

- PNG (recommended for screenshots)
- JPG/JPEG
- GIF
- SVG
